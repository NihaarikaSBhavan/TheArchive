**The Archive**

A lightweight Retrieval-Augmented Generation (RAG) document chat application.
Upload documents, ask questions, and receive grounded AI responses with source citations.

**Features**
- PDF, DOCX, TXT document upload
- Text extraction and chunking
- In-memory similarity search
- Chat interface with history
- Source citations with similarity scores

**Tech Stack**
**Backend:** Python, FastAPI, MongoDB
**Frontend:** React
**LLM:** Gemini 2.5 Flash (via Emergent)
**Embeddings:** Hash-based (demo use)

**Notes**

- Uses in-memory vector storage (non-scalable)
- Built as a learning project
- API usage depends on available credits
